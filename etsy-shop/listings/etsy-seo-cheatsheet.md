*A complete reference guide for optimizing Etsy listings and improving search ranking.*

## Title Structure Formula

### The Formula
**Primary Keyword + Secondary Keyword + Descriptive Feature + Occasion/Use + Style/Material**

### Example Breakdown
```
Handmade Soy Candle | Scented Vanilla Bean | 8oz Mason Jar | Home Decor Gift | Natural Essential Oils
```

| Component | Purpose | Example |
|-----------|---------|---------|
| Primary Keyword | Main search term | "Handmade Soy Candle" |
| Secondary Keyword | Related search | "Scented Vanilla Bean" |
| Descriptive Feature | Specifics buyers want | "8oz Mason Jar" |
| Occasion/Use | When/why they buy | "Home Decor Gift" |
| Style/Material | Differentiator | "Natural Essential Oils" |

### Title Rules
- **Max length:** 140 characters
- **Front-load** your most important keywords
- Use **| - ,** to separate ideas (readability)
- Include **attributes** Etsy's filters use (color, size, occasion)
- Avoid repetition—each word should add value

## Tag Research Strategy (Finding Your 13 Tags)

### Step 1: Brainstorm Seed Keywords
Start with 5-10 words/phrases that describe your product from a buyer's perspective.

### Step 2: Use Etsy's Search Bar
Type your seed keywords and note the autocomplete suggestions—these are real searches.

### Step 3: Analyze Competitors
Search your main keyword → Click top 10 listings → Study their tags using a tool like eRank or Marmalead

### Step 4: Validate with Tools
- **eRank** (free tier available)
- **Marmalead** (paid, more detailed)
- **Everbee** (Chrome extension)

Look for:
- Search volume > 100/month
- Competition "Medium" or lower when possible
- Engagement (favorites/views ratio)

### Step 5: Mix Long-Tail and Short-Tail
| Type | Example | When to Use |
|------|---------|-------------|
| Long-tail (3-4 words) | "personalized wedding gift" | Less competition, higher intent |
| Short-tail (1-2 words) | "candle" | High volume, test if you can rank |

### Tag Best Practices
- Use all **13 tags** every time
- **20 characters max** per tag (including spaces)
- Don't repeat words across tags (wastes space)
- Cover synonyms and variations
- Include plurals if they fit naturally

## Description Optimization Tips

### Structure Template

```
[HOOK - 1-2 sentences grabbing attention]

[BENEFITS - What problem does this solve?]

[FEATURES - Specific details, materials, dimensions]

[PROCESS - How it's made, your story]

[USAGE IDEAS - Gift suggestions, styling tips]

[CARE INSTRUCTIONS - If applicable]

[SHIPPING/POLICY INFO - Set expectations]

[CTA - Encourage purchase]
```

### SEO Tips
- Include your **primary keyword in the first 160 characters**
- Use **natural language**—don't keyword stuff
- **Bold** key information for skimmers
- Add **bullet points** for readability
- Mention **occasions** and **gift recipients**

### Key Elements to Include
- Exact measurements (inches + cm)
- Materials used
- Customization options
- Processing time vs. shipping time
- Return/exchange policy

## Photo SEO Tips

### The 10 Photo Slots Strategy

| Photo # | Purpose | Tips |
|---------|---------|------|
| 1 | Hero shot | Clean background, lifestyle context, bright lighting |
| 2 | Detail/texture | Close-up of materials, craftsmanship |
| 3 | Scale reference | Show size with common objects (coin, hand, ruler) |
| 4 | Variation/options | Different colors, sizes, styles available |
| 5 | Lifestyle in use | Product in its natural environment |
| 6 | Styled flat lay | Aesthetic arrangement, Instagram-worthy |
| 7 | Packaging | Unboxing experience, gift-ready presentation |
| 8 | Process/WIP | Behind-the-scenes, adds authenticity |
| 9 | Bonus/gift idea | Suggests complementary uses |
| 10 | Info graphic | Dimensions, care instructions, features |

### File Naming
Rename photos before upload with descriptive keywords:
```
handmade-leather-wallet-brown-bifold-mens-gift.jpg
```

### Technical Specs
- **Resolution:** 2000px on the shortest side minimum
- **Aspect ratio:** 4:3 recommended
- **Format:** JPG or PNG
- **File size:** Under 20MB per image

## When to Renew Listings

### What Renewal Does
Renewing pushes your listing to the top of search results temporarily—**not** a permanent boost.

### Strategic Timing
| When to Renew | Why |
|---------------|-----|
| After optimizing title/tags | Test improvements faster |
| Seasonal relevance | Before holidays/gifting seasons |
| Low performing but good product | Give it another chance |
| After 60+ days inactive | Recapture visibility |

### Don't Waste Money If...
- Listing has 0 views after 30 days (fix SEO first)
- Product has fundamental issues (price, photos)
- You're in an oversaturated category without differentiation

### Free Alternatives
- Edit listing (small bump)
- Add to new sections/collections
- Share on social media with Etsy link
- Update photos

## Common Mistakes to Avoid

### ❌ Title Mistakes
- Repeating the same words
- Stuffing keywords unnaturally
- Using ALL CAPS
- Including irrelevant terms for "extra visibility"
- Waiting too long to get to the point

### ❌ Tag Mistakes
- Not using all 13 tags
- Using single words when phrases work better
- Repeating words across multiple tags
- Using tags no one searches for
- Ignoring long-tail opportunities

### ❌ Description Mistakes
- Walls of text with no formatting
- Forgetting to include keywords naturally
- No clear sections/bullet points
- Missing size/dimensions
- Unclear shipping times

### ❌ Photo Mistakes
- Blurry or poorly lit images
- No scale reference (guessing sizes)
- Cluttered backgrounds
- Inconsistent editing/filter styles
- Uploading fewer than 5 photos

### ❌ Shop-Level Mistakes
- Incomplete "About" section
- No shop policies set
- Ignoring customer messages
- Inconsistent branding
- Not utilizing Shop Updates
