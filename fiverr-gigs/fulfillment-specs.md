## FULFILLMENT WORKFLOW

1. **Order comes in** → Master E forwards buyer answers to George (Discord)
2. **George delegates** → Scribe generates the deliverable (free via Kimi)
3. **George reviews** → checks quality, formatting, completeness against this spec
4. **Master E reviews** → final check, downloads file
5. **Master E delivers** → uploads to Fiverr, hits Deliver button
6. **Revision requests** → Master E forwards notes, Scribe revises, repeat

### Response time targets:
- New message/inquiry: respond within 1 hour (Fiverr rewards fast replies)
- Order delivery: always deliver 1 day BEFORE the deadline
- Revisions: same day turnaround

### Quality checklist before delivery:
- [ ] Matches the tier the customer paid for
- [ ] All promised features included
- [ ] Spelling/grammar clean
- [ ] Formatted professionally
- [ ] File format correct (PDF/Doc/Sheet)
- [ ] Plagiarism-free (for writing gigs)
